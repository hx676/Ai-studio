## 0. 测试准备

- [ ] 使用新版启动器 `SynCanvasLauncher.exe`，确认标题、图标、品牌名都是 SynCanvas。
- [ ] 准备临时测试素材：1 张图片、1 段短音频、1 段短视频，删除测试只用临时素材。

## 1. 启动器

- [ ] 首页显示主应用、TTS、HeyGem 三后台状态。
- [ ] “一键启动”能冷启动主应用、TTS、HeyGem。
- [ ] 启动过程中控制台 1 秒内出现：正在启动、端口、PID、就绪状态。
- [ ] 三后台 ready 后再次点“一键启动”，显示复用，不重复启动 TTS/HeyGem。
- [ ] “打开主应用”打开 `http://127.0.0.1:3000/`，进入前端首页。
- [ ] 默认一键启动完成后不自动弹浏览器。
- [ ] 设置里勾选“启动后自动打开主应用”后，启动完成会自动打开首页。
- [ ] “一键停止”能关闭本次启动器拉起的主应用、TTS、HeyGem。
- [ ] 直接关闭启动器后，后台进程也被清理。
- [ ] 控制台能追加显示关键日志，不显示健康检查、模型加载细节、媒体 206 等噪音。
- [ ] “清空屏幕”只清 UI，不删除日志文件。
- [ ] “打开日志目录”可用。
- [ ] “导出完整日志”包含原始完整日志，不做前端过滤。
- [ ] 疑难解答只显示需要处理的问题，正常项不刷屏。
- [ ] 设置页保存后重启启动器仍生效。
- [ ] 左侧导航不再有“小工具”。
